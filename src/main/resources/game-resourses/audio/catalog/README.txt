AUDIO CATALOG

This folder does not replace the original resource paths.
It is only a sorted view of what is currently used in code.

Folders:
- used/menu/background
- used/menu/ambience
- used/ui/menu
- used/ui/system
- used/level/background
- used/level/ambience
- used/level/results

Files:
- USED_IN_CODE.txt
  Human-readable grouped list of current sounds/music in use.

- UNUSED_NOT_IN_CODE.txt
  Original audio files that exist in resources but are not currently referenced in code.

- used_paths.txt
  Raw source paths that are currently used.

- all_original_audio_paths.txt
  Raw list of all original audio files under src/main/resources/game-resourses/audio
  excluding this catalog and the old curated helper folder.
